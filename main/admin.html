Admin

Add tracks and events. New tracks are auto-enriched from Wikidata, Wikimedia Commons and OpenStreetMap.

Add a Track

Type a venue name and search — coordinates, photo, website, opening year and location details are filled in automatically where available. Everything stays editable before staging.

Discovered Venues

UK venues the data pipeline found on Wikidata that aren't on the site yet (includes historic circuits — accept only ones that belong). Auto-add pulls everything — location, photo, race types, a summary from Wikipedia — and stages the track with no manual input (venues that look closed/former are refused and need Review). Review opens the editable form instead. Once committed, the nightly refresh removes the venue from this list automatically.

Add an Event

For one-off events or series without a scraper yet. Scraped series (BTCC, BSB, British GT, Hillclimb, Santa Pod) are replaced by the nightly refresh — manual events for those series will be overwritten.

Publish Changes

Nothing staged yet.

Committing pushes the updated data to GitHub, which redeploys the live site automatically (about a minute). The token is used only for these requests and never stored — a fine-grained token with Contents: read & write on the RaceDates repo is enough.
